\# Knowledge Tracker



A personal knowledge tracking web application to help organize, monitor, and revisit learning topics effectively.



\## Overview



The Knowledge Tracker treats each topic or concept as a structured knowledge unit instead of plain notes.  

It helps identify weak areas, track revision status, and maintain clarity over ongoing learning.



\## Features



\- Add and manage knowledge units

\- Categorize topics (DSA, Web, ML, etc.)

\- Track learning status and confidence level

\- Filter topics by category, difficulty, or status

\- Simple and clean UI for focused usage
